# Vote Ready 2026

A single-page static site helping Americans — especially low-income and vulnerable citizens — prepare to vote in the November 2026 elections. It explains new voter ID requirements under the SAVE America Act and walks readers through a clear, seven-step action timeline to get registered, get their documents, and get to the polls.

**Live site:** [voteready2026.pages.dev](https://voteready2026.pages.dev)

## Run locally

Open `index.html` in any browser. No build step or server required.

For live reload during development, use any static file server:

```sh
npx serve .
```
